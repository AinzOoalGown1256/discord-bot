# Discord Bot - Salas Temporales

Bot en Python usando nextcord que crea salas de voz y texto temporales automáticamente.

## Características
- Slash commands: /setup, /yp generador
- Crea canales al unirse a uno generador
- Elimina los canales cuando están vacíos

## Deploy 24/7 en Fly.io
- Dockerfile incluido
- Variables de entorno con flyctl secrets
